Задача: Создать игру в жанре стратегии, основанную на реальных картах (например, Google Maps). Игра должна включать следующие элементы:

Игровой мир:

Использовать реальные карты (Google Maps, OpenStreetMap или аналогичные) для создания игрового поля.

Игроки начинают игру с одного захваченного города (столицы), который является их стартовой базой. Город можно выбрать в начале игры.

Каждый город на карте имеет уникальные характеристики и преимущества (например, ресурсы: золото, нефть, древесина, еда и т.д.). Город так же имеет границы картографические.

Ресурсы и здания:

Каждый город имеет бонусы, например Москва может приносить +10 дерева если будет построена лесорубка. требуемые ресурсы для постройки зданий придумай сам согласно балансу.

Игрок начинает с начальными ресурсами, например по 500 золота, нефти, еды и дерева

Для добычи ресурсов необходимо строить здания (например, шахты для золота, нефтяные вышки для нефти, фермы для еды).

У каждого города есть ограничение по населению, которое влияет на количество доступных рабочих и солдат. Чем больше город по размеру тем больше население в нем может быть. Например у москвы это 150 тысяч человек.

У каждого города есть свои ресурсы. Например у москвы +10 дерева(требуется постройка лесорубки для добычи) Это необходимо обозначить. Например под названием города будет иконка ресурса и то какое количество он может добывать. Ресурс добывается каждую секунду.

Население:

Население — это ключевой ресурс, который можно использовать для строительства, добычи ресурсов или атаки на другие города.

Население растет со временем
(игрок начинает с населением 0, но с каждой секундной оно увеличивается на 1), но его можно увеличить, строя жилые здания или захватывая новые города.

Боевая система:

Игроку необходимо построить бараки для создании армии. Для барак необходимо 700 дерева и будет производиться 50 солдат в минуту.

Для нейтральной территории требуется солдат 1/4 от всего населения нейтральной территории
После успешной атаки город переходит под контроль игрока и начинает производить ресурсы.

Дипломатия и альянсы:

Игроки могут заключать союзы, обмениваться ресурсами или объединяться против общего врага.

Возможность торговли ресурсами между игроками.


Технические требования:

Использовать API карт (например, leaflet openstreetmap) для отображения игрового поля.

Реализовать систему управления ресурсами, населением и зданиями.

Создать интерфейс для управления игрой (строительство, атаки, дипломатия).

Добавить мультиплеерный режим, чтобы игроки могли играть друг против друга.

Дизайн:

Захваченная территория обозначается синей, нейтральная серой, вражеская красной, а союзная зеленой. Должна быть область и границы у территорий.

У каждого ресурса есть своя иконка. 

Реализуй все города России и обозначь для каждого города границы

Главный интерфейс отвечает за показ ресурсов игрока и то какой город он нажал, его характеристики и возможность постройки. Этот интерфейс должен быть выше по слоям чем карта(я заметил у тебя есть проблемы с отображением этого слоя ибо оно показывается только когда приближаешь или отдаляешь карту)
